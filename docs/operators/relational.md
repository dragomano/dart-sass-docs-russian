> Операторы сравнения определяют, являются ли [числа](../values/numbers) больше или меньше друг друга. Они автоматически выполняют преобразование между совместимыми единицами измерения.

* `<выражение> < <выражение>` возвращает, меньше ли значение первого [выражения](../syntax/structure#expressions), чем второго.
* `<выражение> <= <выражение>` возвращает, меньше ли или равно значение первого [выражения](../syntax/structure#expressions) значению второго.
* `<выражение> > <выражение>` возвращает, больше ли значение первого [выражения](../syntax/structure#expressions), чем второго.
* `<выражение> >= <выражение>` возвращает, больше ли или равно значение первого [выражения](../syntax/structure#expressions) значению второго.

=== "SCSS"

    ```scss
    @debug 100 > 50; // true
    @debug 10px < 17px; // true
    @debug 96px >= 1in; // true
    @debug 1000ms <= 1s; // true
    ```

=== "SASS"

    ```sass
    @debug 100 > 50 // true
    @debug 10px < 17px // true
    @debug 96px >= 1in // true
    @debug 1000ms <= 1s // true
    ```

Безразмерные числа можно сравнивать с любыми числами. Они автоматически преобразуются в единицу измерения этого числа.

=== "SCSS"

    ```scss
    @debug 100 > 50px; // true
    @debug 10px < 17; // true
    ```

=== "SASS"

    ```sass
    @debug 100 > 50px // true
    @debug 10px < 17 // true
    ```

Числа с несовместимыми единицами нельзя сравнивать.

=== "SCSS"

    ```scss
    @debug 100px > 10s;
    //     ^^^^^^^^^^^
    // Ошибка: несовместимые единицы измерения px и s.
    ```

=== "SASS"

    ```sass
    @debug 100px > 10s
    //     ^^^^^^^^^^^
    // Ошибка: несовместимые единицы измерения px и s.
    ```

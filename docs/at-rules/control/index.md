title: Правила управления потоком

> Sass предоставляет ряд at-правил, которые позволяют управлять тем, будут ли стили выводиться, а также выводить их несколько раз с небольшими вариациями. Их также можно использовать в [миксинах](../at-rules/mixin) и [функциях](../at-rules/function) для написания небольших алгоритмов, которые упрощают работу с Sass. Sass поддерживает четыре правила управления потоком.

- [`@if`](../at-rules/control/if) управляет тем, будет ли блок вычисляться или нет.

- [`@each`](../at-rules/control/each) вычисляет блок для каждого элемента в [списке](../values/lists) или каждой пары в [карте](../values/maps).

- [`@for`](../at-rules/control/for) вычисляет блок заданное количество раз.

- [`@while`](../at-rules/control/while) вычисляет блок до тех пор, пока выполняется определённое условие.

## –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ {#usage}

–ò—Å–ø–æ–ª–Ω—è–µ–º—ã–π —Ñ–∞–π–ª Dart Sass –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø—É—â–µ–Ω –≤ –æ–¥–Ω–æ–º –∏–∑ –¥–≤—É—Ö —Ä–µ–∂–∏–º–æ–≤.

### –†–µ–∂–∏–º ¬´–æ–¥–∏–Ω –∫ –æ–¥–Ω–æ–º—É¬ª {#one-to-one-mode}


–†–µ–∂–∏–º ¬´–æ–¥–∏–Ω –∫ –æ–¥–Ω–æ–º—É¬ª –∫–æ–º–ø–∏–ª–∏—Ä—É–µ—Ç –æ–¥–∏–Ω –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª (`input.scss`) –≤ –æ–¥–Ω–æ –≤—ã—Ö–æ–¥–Ω–æ–µ —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏–µ (`output.css`). –ï—Å–ª–∏ –≤—ã—Ö–æ–¥–Ω–æ–µ —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏–µ –Ω–µ —É–∫–∞–∑–∞–Ω–æ, —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–Ω—ã–π CSS –≤—ã–≤–æ–¥–∏—Ç—Å—è –≤ —Ç–µ—Ä–º–∏–Ω–∞–ª.

–í—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª –ø–∞—Ä—Å–∏—Ç—Å—è –∫–∞–∫ [SCSS](../syntax#scss), –µ—Å–ª–∏ –µ–≥–æ —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ `.scss`, –∫–∞–∫ [—Å–∏–Ω—Ç–∞–∫—Å–∏—Å —Å –æ—Ç—Å—Ç—É–ø–∞–º–∏](../syntax#the-indented-syntax), –µ—Å–ª–∏ –µ–≥–æ —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ `.sass`, –∏–ª–∏ –∫–∞–∫ [–ø—Ä–æ—Å—Ç–æ–π CSS](../at-rules/import/#importing-css), –µ—Å–ª–∏ –µ–≥–æ —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ `.css`. –ï—Å–ª–∏ —É –Ω–µ–≥–æ –Ω–µ—Ç –æ–¥–Ω–æ–≥–æ –∏–∑ —ç—Ç–∏—Ö —Ç—Ä—ë—Ö —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–π –∏–ª–∏ –µ—Å–ª–∏ –æ–Ω –ø–æ—Å—Ç—É–ø–∞–µ—Ç –∏–∑ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –≤–≤–æ–¥–∞, –æ–Ω –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –ø–∞—Ä—Å–∏—Ç—Å—è –∫–∞–∫ SCSS. –≠—Ç–æ –º–æ–∂–Ω–æ –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä–æ–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ–ª–∞–≥–∞ [`--indented`](#indented).

–°–ø–µ—Ü–∏–∞–ª—å–Ω–∞—è —Å—Ç—Ä–æ–∫–∞ `-` –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–µ—Ä–µ–¥–∞–Ω–∞ –≤–º–µ—Å—Ç–æ –≤—Ö–æ–¥–Ω–æ–≥–æ —Ñ–∞–π–ª–∞, —á—Ç–æ–±—ã —É–∫–∞–∑–∞—Ç—å Sass —á–∏—Ç–∞—Ç—å –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª –∏–∑ [—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –≤–≤–æ–¥–∞](https://en.wikipedia.org/wiki/Standard_streams#Standard_input_(stdin)). Sass –±—É–¥–µ—Ç –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –ø–∞—Ä—Å–∏—Ç—å –µ–≥–æ –∫–∞–∫ SCSS, –µ—Å–ª–∏ –Ω–µ –ø–µ—Ä–µ–¥–∞–Ω —Ñ–ª–∞–≥ [`--indented`](#indented).

### –†–µ–∂–∏–º ¬´–º–Ω–æ–≥–∏–µ –∫–æ –º–Ω–æ–≥–∏–º¬ª {#many-to-many-mode}


–†–µ–∂–∏–º ¬´–º–Ω–æ–≥–∏–µ –∫–æ –º–Ω–æ–≥–∏–º¬ª –∫–æ–º–ø–∏–ª–∏—Ä—É–µ—Ç –æ–¥–∏–Ω –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ –≤—Ö–æ–¥–Ω—ã—Ö —Ñ–∞–π–ª–æ–≤ –≤ –æ–¥–∏–Ω –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ –≤—ã—Ö–æ–¥–Ω—ã—Ö —Ñ–∞–π–ª–æ–≤. –í—Ö–æ–¥–Ω—ã–µ —Ñ–∞–π–ª—ã –æ—Ç–¥–µ–ª—è—é—Ç—Å—è –æ—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–≤–æ–µ—Ç–æ—á–∏—è–º–∏. –û–Ω —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç –∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞—Ç—å –≤—Å–µ —Ñ–∞–π–ª—ã Sass –≤ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –≤ —Ñ–∞–π–ª—ã CSS —Å —Ç–µ–º–∏ –∂–µ –∏–º–µ–Ω–∞–º–∏ –≤ –¥—Ä—É–≥–æ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏.

–ü—Ä–∏ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ —Ü–µ–ª—ã—Ö –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–π Sass –∏–≥–Ω–æ—Ä–∏—Ä—É–µ—Ç [—Ñ—Ä–∞–≥–º–µ–Ω—Ç—ã](../at-rules/import/#partials), –∏–º–µ–Ω–∞ –∫–æ—Ç–æ—Ä—ã—Ö –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å `_`. –í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ—Ä–∞–≥–º–µ–Ω—Ç—ã –¥–ª—è —Ä–∞–∑–¥–µ–ª–µ–Ω–∏—è —Ç–∞–±–ª–∏—Ü —Å—Ç–∏–ª–µ–π –±–µ–∑ —Å–æ–∑–¥–∞–Ω–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –Ω–µ–Ω—É–∂–Ω—ã—Ö –≤—ã—Ö–æ–¥–Ω—ã—Ö —Ñ–∞–π–ª–æ–≤.

## –û–ø—Ü–∏–∏ {#options}

### –í–≤–æ–¥ –∏ –≤—ã–≤–æ–¥ {#input-and-output}

#### `--stdin` {#stdin}

–≠—Ç–æ—Ç —Ñ–ª–∞–≥ —è–≤–ª—è–µ—Ç—Å—è –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω—ã–º —Å–ø–æ—Å–æ–±–æ–º —É–∫–∞–∑–∞—Ç—å Sass, —á—Ç–æ –æ–Ω –¥–æ–ª–∂–µ–Ω —á–∏—Ç–∞—Ç—å –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª –∏–∑ [—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –≤–≤–æ–¥–∞](https://en.wikipedia.org/wiki/Standard_streams#Standard_input_(stdin)). –ö–æ–≥–¥–∞ –æ–Ω –ø–µ—Ä–µ–¥–∞–Ω, –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å —É–∫–∞–∑–∞–Ω.

–§–ª–∞–≥ `--indented` –Ω–∞–∏–±–æ–ª–µ–µ –ø–æ–ª–µ–∑–µ–Ω, –∫–æ–≥–¥–∞ –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª –ø–æ—Å—Ç—É–ø–∞–µ—Ç –∏–∑ [—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –≤–≤–æ–¥–∞](https://en.wikipedia.org/wiki/Standard_streams#Standard_input_(stdin)), –ø–æ—ç—Ç–æ–º—É –µ–≥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏.

#### `--style` {#style}

#### `--no-charset` {#no-charset}

–≠—Ç–æ—Ç —Ñ–ª–∞–≥ —É–∫–∞–∑—ã–≤–∞–µ—Ç Sass –Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –≤—ã–≤–æ–¥–∏—Ç—å –æ–±—ä—è–≤–ª–µ–Ω–∏–µ `@charset` –∏–ª–∏ UTF-8 [–º–µ—Ç–∫—É –ø–æ—Ä—è–¥–∫–∞ –±–∞–π—Ç–æ–≤](https://en.wikipedia.org/wiki/Byte_order_mark#UTF-8). –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é, –∏–ª–∏ –µ—Å–ª–∏ –ø–µ—Ä–µ–¥–∞–Ω `--charset`, Sass –≤—Å—Ç–∞–≤–∏—Ç –ª–∏–±–æ –æ–±—ä—è–≤–ª–µ–Ω–∏–µ `@charset` (–≤ —Ä–µ–∂–∏–º–µ —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω–æ–≥–æ –≤—ã–≤–æ–¥–∞), –ª–∏–±–æ –º–µ—Ç–∫—É –ø–æ—Ä—è–¥–∫–∞ –±–∞–π—Ç–æ–≤ (–≤ —Ä–µ–∂–∏–º–µ —Å–∂–∞—Ç–æ–≥–æ –≤—ã–≤–æ–¥–∞), –µ—Å–ª–∏ —Ç–∞–±–ª–∏—Ü–∞ —Å—Ç–∏–ª–µ–π —Å–æ–¥–µ—Ä–∂–∏—Ç —Å–∏–º–≤–æ–ª—ã, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç ASCII.

```sh

  content: "üë≠";
}
```
